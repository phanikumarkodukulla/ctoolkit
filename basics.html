Introduction to C

History of C

C was developed by Dennis Ritchie at Bell Laboratories in the early 1970s. It was designed as a system programming language for the UNIX operating system. Prior to C, most operating systems were written in assembly language. C was developed to provide a higher-level alternative that still allowed low-level memory access and efficient code generation.

Features of C

  • Mid-level language: C combines elements of both high-level and low-level languages
  • Portability: Programs written in C can be run on different platforms with minimal or no modifications
  • Modularity: C supports structured programming through functions
  • Efficiency: C programs are fast and memory-efficient
  • Flexibility: C can be used for various applications, from system software to application software
  • Rich set of operators: C provides a wide range of operators for arithmetic, logical, and bitwise operations
  • Memory management: C allows dynamic memory allocation through functions like malloc() and free()

Importance of C in Programming

C has played a pivotal role in the development of modern computing. Many reasons make C an important language:

  • Most operating systems (like UNIX, Linux) are written in C
  • C is the foundation for many other programming languages like C++, Java, Python, etc.
  • Used extensively in embedded systems programming
  • Performance-critical applications often use C for its speed and efficiency
  • Learning C provides a strong understanding of memory management and computer architecture

Setting Up the Environment

Installation of C Compiler (GCC)

To write and run C programs, you need a compiler. The GNU Compiler Collection (GCC) is one of the most popular C compilers available.

On Windows:

  1. Install MinGW (Minimalist GNU for Windows) or Cygwin
  2. Add the bin directory to your PATH environment variable
  3. Open command prompt and type gcc --version to verify installation

On Linux:

sudo apt-get update
sudo apt-get install gcc

On macOS:

xcode-select --install

Writing and Compiling a Simple C Program

Let's create a simple "Hello, World!" program to test our setup:

// hello.c
#include <stdio.h>

int main() {
    printf("Hello, World!\n");
    return 0;
}

To compile and run this program:

gcc hello.c -o hello
./hello
Hello, World!

This simple example demonstrates the basic workflow of writing, compiling, and running a C program.

Basic Structure of C Program

A C program consists of various components that work together to perform a task. Here's the basic structure:

// Documentation Section
/* This program demonstrates the basic structure of a C program */

// Link Section
#include <stdio.h>  // Standard Input/Output header file

// Definition Section
#define PI 3.14159

// Global Declaration Section
int globalVar;  // Global variable declaration

// Function Declaration Section
int add(int a, int b);  // Function prototype

// Main Function Section
int main() {
    // Local Variable Declaration
    int sum;
    
    // Statements & Expressions
    sum = add(5, 10);
    printf("Sum: %d\n", sum);
    
    // Return Statement
    return 0;
}

// Function Definition Section
int add(int a, int b) {
    return a + b;
}

Header Files (#include)

Header files contain function declarations and macro definitions. The #include directive tells the preprocessor to insert the contents of the specified file into the program.

Common header files:

  • <stdio.h> - Standard Input/Output functions like printf(), scanf()
  • <stdlib.h> - Standard Library functions like malloc(), free()
  • <string.h> - String manipulation functions like strcpy(), strlen()
  • <math.h> - Mathematical functions like sin(), cos(), sqrt()

Main Function (int main())

The main() function is the entry point of a C program. Program execution begins from this function. It can have two forms:

// Form 1: No command line arguments
int main() {
    // Statements
    return 0;
}

// Form 2: With command line arguments
int main(int argc, char *argv[]) {
    // argc: argument count
    // argv: argument vector (array of strings)
    return 0;
}

Return Statements (return 0;)

The return statement in the main function indicates the status of program execution to the operating system.

  • return 0; - Successful execution
  • return non-zero; - Indicates an error occurred

Data Types

C language provides various data types to define variables and functions. The data types in C can be classified into three categories:

Primitive Data Types

Type Size (typically) Range Example

Type modifiers can be used with these data types:

Derived Data Types

Derived data types are derived from primitive data types:

Arrays

An array is a collection of elements of the same data type.

int numbers[5] = {10, 20, 30, 40, 50}; // Array of integers
char name[10] = "C Program"; // Array of characters (string)

Pointers

A pointer is a variable that stores the memory address of another variable.

int num = 10;
int *ptr = # // ptr stores the address of num

Structures

A structure is a collection of variables of different data types under a single name.

struct Student {
    char name[50];

struct Student s1 = {"John", 101, 85.5};

User-defined Data Types (typedef)

The typedef keyword allows you to create your own data type names:

    int x;

Variables and Constants

Declaration of Variables

In C, variables must be declared before they can be used. The declaration specifies the data type and name of the variable.

int age; // Declaration
float salary;
char initial;

Variables can also be declared in a single line if they share the same data type:

int a, b, c; // Multiple declarations

Assigning Values to Variables

Values can be assigned to variables at declaration time or later in the program:

int count = 10; // Declaration with initialization
float pi; // Declaration only
pi = 3.14159; // Assignment later

Multiple assignments can be done in a single line:

int a = 5, b = 10, c = 15; // Multiple initializations

Variable values can be changed during program execution:

int x = 10;
x = 20; // x now contains 20
x = x + 5; // x now contains 25

Constants

Constants are fixed values that cannot be altered during program execution. There are two ways to define constants in C:

Using #define (Preprocessor Directive)

#define PI 3.14159
#define MAX_SIZE 100

float area = PI * radius * radius;

The preprocessor replaces all occurrences of PI with 3.14159 before compilation.

Using const Keyword

const float PI = 3.14159;
const int MAX_SIZE = 100;

const char* MESSAGE = "Error occurred!";

When using const, the variable is stored in memory but cannot be modified.

Variable Scope

The scope of a variable determines where it can be accessed in a program:

// Global variable (file scope)
int globalVar = 100;

void function() {
    // Local variable (function scope)
    int localVar = 10;
    
    {
        // Block scope variable
        int blockVar = 5;
    }
    
    // blockVar cannot be accessed here
}

Best Practices for Variables and Constants

  • Use meaningful variable names
  • Initialize variables when declaring them
  • Use constants for values that should not change
  • Keep variable scope as limited as possible

Input and Output

Input and output operations are essential parts of most programs. In C, these operations are primarily handled by functions in the standard I/O library (<stdio.h>).

Output with printf()

The printf() function is used to display formatted output to the screen:

printf("Hello, World!\n"); // Simple string output

int num = 10;
printf("The value is %d\n", num); // Format specifier for integer

float pi = 3.14159;
printf("Pi is approximately %.2f\n", pi); // Format specifier with precision
Hello, World!
The value is 10
Pi is approximately 3.14

Common Format Specifiers

Specifier Output Example
%d or %i Signed integer printf("%d", 10);
%u Unsigned integer printf("%u", 100);
%f Float or double printf("%f", 3.14);
%c Character printf("%c", 'A');
%s String printf("%s", "Hello");
%p Pointer address printf("%p", &var);

Format Specifier Modifiers

printf("%5d", 10);     // Right-justified, width 5
printf("%-5d", 10);    // Left-justified, width 5
printf("%05d", 10);    // Zero-padded, width 5
printf("%.2f", 3.14159); // Precision 2 for float
10
10
00010

Input with scanf()

The scanf() function is used to read formatted input from the keyboard:

int age;
printf("Enter your age: ");
scanf("%d", &age);
printf("You are %d years old.\n", age);

Note the & operator before the variable name. This provides the memory address where scanf() should store the input value.

Reading Multiple Values

int x, y;
printf("Enter two numbers separated by a space: ");
scanf("%d %d", &x, &y);
printf("Sum: %d\n", x + y);

Reading Strings

char name[50];
printf("Enter your name: ");
scanf("%s", name); // No & operator needed for arrays
printf("Hello, %s!\n", name);

Note: scanf() with %s stops reading at the first whitespace character. To read a full line including spaces, use fgets():

char fullName[50];
printf("Enter your full name: ");
fgets(fullName, 50, stdin);
printf("Hello, %s", fullName);

Other Input/Output Functions

  • putchar() - Outputs a single character
  • getchar() - Reads a single character
  • puts() - Outputs a string followed by a newline
  • gets() - Reads a line (deprecated due to safety issues)
  • fprintf() - Writes formatted output to a file
  • fscanf() - Reads formatted input from a file

int a = 10, b = 3;

Relational Operators

Used to compare values:

Operator Description Example Result
== Equal to 5 == 5 1 (true)
!= Not equal to 5 != 3 1 (true)
> Greater than 5 > 3 1 (true)
< Less than 5 < 3 0 (false)
>= Greater than or equal to 5 >= 5 1 (true)
<= Less than or equal to 5 <= 3 0 (false)

Example:

int a = 5, b = 10;
if (a == b) {
    printf("a equals b\n");
} else if (a < b) {
    printf("a is less than b\n");
} else {
    printf("a is greater than b\n");
}
a is less than b

Logical Operators

Used to combine multiple conditions:

Operator Description Example Result
&& Logical AND (5 > 3) && (10 > 5) 1 (true)
|| Logical OR (5 < 3) || (10 > 5) 1 (true)
! Logical NOT !(5 < 3) 1 (true)

Example:

int age = 25;
int income = 50000;

if (age > 18 && income > 30000) {
    printf("Eligible for credit card\n");
}

if (age < 18 || income < 20000) {
    printf("Not eligible for loan\n");
}

if (!(age < 18)) {
    printf("Adult\n");
}
Eligible for credit card
Adult

Assignment Operators

Used to assign values to variables:

Operator Description Example Equivalent to
= Simple assignment a = 5 -
+= Add and assign a += 3 a = a + 3
-= Subtract and assign a -= 3 a = a - 3
*= Multiply and assign a *= 3 a = a * 3
/= Divide and assign a /= 3 a = a / 3
%= Modulus and assign a %= 3 a = a % 3

Example:

int x = 10;
x += 5;  // x is now 15
x -= 3;  // x is now 12
x *= 2;  // x is now 24
x /= 4;  // x is now 6
x %= 4;  // x is now 2

Bitwise Operators

Used to perform bit-level operations on binary representations of numbers:

Operator Description Example Result
& Bitwise AND 5 & 3 (0101 & 0011) 1 (0001 in binary)
| Bitwise OR 5 | 3 (0101 | 0011) 7 (0111 in binary)
^ Bitwise XOR 5 ^ 3 (0101 ^ 0011) 6 (0110 in binary)
~ Bitwise NOT ~5 (~0101) -6 (depends on system)
<< Left shift 5 << 1 (0101 << 1) 10 (1010 in binary)
>> Right shift 5 >> 1 (0101 >> 1) 2 (0010 in binary)

Example:

unsigned char a = 5;  // 00000101 in binary
unsigned char b = 3;  // 00000011 in binary

printf("a & b = %d\n", a & b);   // Bitwise AND: 00000001 = 1
printf("a | b = %d\n", a | b);   // Bitwise OR: 00000111 = 7
printf("a ^ b = %d\n", a ^ b);   // Bitwise XOR: 00000110 = 6
printf("~a = %d\n", ~a);         // Bitwise NOT: 11111010 = 250 (unsigned)
printf("a << 1 = %d\n", a << 1); // Left shift: 00001010 = 10
printf("a >> 1 = %d\n", a >> 1); // Right shift: 00000010 = 2

Increment and Decrement Operators

Used to increase or decrease the value of a variable by 1:

Operator Description Example
++ Increment by 1 ++a or a++
-- Decrement by 1 --a or a--

The position of the operator matters:

  • Prefix (++a): Increment first, then use the value
  • Postfix (a++): Use the value first, then increment

Example:

int a = 5;
int b, c;

b = ++a;  // a is incremented to 6, then b gets the value 6
printf("a = %d, b = %d\n", a, b);  // a = 6, b = 6

a = 5;    // Reset a
c = a++;  // c gets the current value of a (5), then a is incremented to 6
printf("a = %d, c = %d\n", a, c);  // a = 6, c = 5

Conditional (Ternary) Operator

A shorthand version of the if-else statement:

// Syntax: condition ? expression_if_true : expression_if_false

int a = 10, b = 20;
int max = (a > b) ? a : b;  // max will be 20

// Equivalent if-else:
int max;
if (a > b)
    max = a;
else
    max = b;

Operator Precedence

Operators have different priorities that determine the order of evaluation in an expression:

  1. Postfix ((), [], ->, .) and Unary (!, ~, ++, --, +, -, * (dereference), & (address-of))
  2. Multiplicative (*, /, %)
  3. Additive (+, -)
  4. Shift (<<, >>)
  5. Relational (<, <=, >, >=)
  6. Equality (==, !=)
  7. Bitwise AND (&)
  8. Bitwise XOR (^)
  9. Bitwise OR (|)
  10. Logical AND (&&)
  11. Logical OR (||)
  12. Conditional (?:)
  13. Assignment (=, +=, -=, etc.)
  14. Comma (,)

When in doubt, use parentheses to enforce the order of operations:

int result = 5 + 3 * 2;     // result = 11 (multiplication happens first)
int result2 = (5 + 3) * 2;  // result2 = 16 (addition happens first due to parentheses)

Control Statements

Control statements allow you to control the flow of program execution based on certain conditions or to repeat a block of statements multiple times.

Conditional Statements

if Statement

The if statement executes a block of code if a specified condition is true:

int age = 20;

if (age >= 18) {
    printf("You are eligible to vote.\n");
}
You are eligible to vote.

if-else Statement

The if-else statement executes one block of code if a condition is true and another block if the condition is false:

int number = 15;

if (number % 2 == 0) {
    printf("%d is even.\n", number);
} else {
    printf("%d is odd.\n", number);
}
15 is odd.

if-else if-else Statement

The if-else if-else statement checks multiple conditions in sequence:

int score = 85;

if (score >= 90) {
    printf("Grade: A\n");
} else if (score >= 80) {
    printf("Grade: B\n");
} else if (score >= 70) {
    printf("Grade: C\n");
} else if (score >= 60) {
    printf("Grade: D\n");
} else {
    printf("Grade: F\n");
}
Grade: B

Nested if Statements

You can place an if statement inside another if statement:

int age = 25;
int income = 50000;

if (age > 18) {
    if (income > 30000) {
        printf("Eligible for premium credit card.\n");
    } else {
        printf("Eligible for basic credit card.\n");
    }
} else {
    printf("Not eligible for credit card.\n");
}
Eligible for premium credit card.

switch Statement

The switch statement selects one of many code blocks to be executed:

int day = 4;

switch (day) {
    case 1:
        printf("Monday\n");
        break;
    case 2:
        printf("Tuesday\n");
        break;
    case 3:
        printf("Wednesday\n");
        break;
    case 4:
        printf("Thursday\n");
        break;
    case 5:
        printf("Friday\n");
        break;
    case 6:
        printf("Saturday\n");
        break;
    case 7:
        printf("Sunday\n");
        break;
    default:
        printf("Invalid day number\n");
}
Thursday

Notes on switch statement:

  • The expression in switch must be an integer or character type
  • The break statement is used to exit the switch block
  • If break is omitted, execution continues to the next case
  • The default case is executed if no match is found

Looping Statements

Loops are used to execute a block of code repeatedly.

for Loop

The for loop is used when you know in advance how many times you want to execute a statement or block of statements:

// Syntax: for (initialization; condition; increment/decrement)

// Print numbers from 1 to 5
for (int i = 1; i <= 5; i++) {
    printf("%d ", i);
}
printf("\n");
1 2 3 4 5

while Loop

The while loop executes a block of code as long as a specified condition is true:

// Syntax: while (condition) { statements; }

int i = 1;
while (i <= 5) {
    printf("%d ", i);
    i++;
}
printf("\n");
1 2 3 4 5

do-while Loop

The do-while loop is similar to the while loop, but it executes the code block once before checking the condition:

// Syntax: do { statements; } while (condition);

int i = 1;
do {
    printf("%d ", i);
    i++;
} while (i <= 5);
printf("\n");
1 2 3 4 5

The key difference between while and do-while is that do-while guarantees that the code block will execute at least once, even if the condition is initially false.

Nested Loops

You can place one loop inside another loop:

// Print a 3x3 pattern of asterisks
for (int i = 0; i < 3; i++) {
    for (int j = 0; j < 3; j++) {
        printf("* ");
    }
    printf("\n");
}
* * *
* * *
* * *

Jump Statements

Jump statements transfer control to another part of the program.

break Statement

The break statement terminates the innermost loop or switch statement:

// Find the first even number in the range
for (int i = 1; i <= 10; i++) {
    if (i % 2 == 0) {
        printf("First even number: %d\n", i);
        break;  // Exit the loop when first even number is found
    }
}
First even number: 2

continue Statement

The continue statement skips the rest of the current iteration and continues with the next iteration:

// Print odd numbers from 1 to 10
for (int i = 1; i <= 10; i++) {
    if (i % 2 == 0) {
        continue;  // Skip even numbers
    }
    printf("%d ", i);
}
printf("\n");
1 3 5 7 9

goto Statement

The goto statement transfers control to a labeled statement within the same function:

int i = 1;

start:  // Label
    if (i <= 5) {
        printf("%d ", i);
        i++;
        goto start;  // Jump to the label
    }
    printf("\n");
1 2 3 4 5

Note: The goto statement is generally discouraged in modern programming practices because it can make code difficult to understand and maintain.

Functions

Functions are self-contained blocks of code that perform a specific task. They help in organizing code, improving reusability, and making programs easier to maintain.

Function Declaration and Definition

A function in C has the following general form:

return_type function_name(parameter_list) {
    // function body
    statements;
    return value; // optional
}
  • return_type: The data type of the value returned by the function (or void if no value is returned)
  • function_name: The name of the function
  • parameter_list: A comma-separated list of parameters (or void if no parameters)
  • function body: The block of statements that define what the function does

Example:

// Function to calculate the sum of two integers
int add(int a, int b) {
    int result = a + b;
    return result;
}

Function Declaration vs. Definition

  • Function Declaration (also called function prototype): Tells the compiler about the function name, return type, and parameters.
  • Function Definition: Contains the actual implementation of the function.
// Function declaration (prototype)
int add(int, int);  // Parameter names are optional in declaration

// Function definition
int add(int a, int b) {
    return a + b;
}

Function Call

To use a function, you need to call it by its name and provide the required arguments:

int main() {
    int x = 5, y = 10;
    int sum = add(x, y);  // Function call
    printf("Sum: %d\n", sum);
    return 0;
}
Sum: 15

Function Parameters

1. Call by Value

When you pass arguments by value, a copy of the value is passed to the function. Changes to the parameter inside the function do not affect the original variable.

void increment(int x) {
    x++;  // Only changes the local copy
    printf("Inside function: x = %d\n", x);
}

int main() {
    int num = 5;
    printf("Before function call: num = %d\n", num);
    increment(num);
    printf("After function call: num = %d\n", num);
    return 0;
}
Before function call: num = 5
Inside function: x = 6
After function call: num = 5

2. Call by Reference

When you pass arguments by reference (using pointers), the function can modify the original variable.

void increment(int *x) {
    (*x)++;  // Changes the value at the address
    printf("Inside function: *x = %d\n", *x);
}

int main() {
    int num = 5;
    printf("Before function call: num = %d\n", num);
    increment(&num);  // Pass the address of num
    printf("After function call: num = %d\n", num);
    return 0;
}
Before function call: num = 5
Inside function: *x = 6
After function call: num = 6

Function Return Values

Functions can return a value using the return statement. If a function doesn't return a value, its return type should be void.

// Function returning an integer
int square(int x) {
    return x * x;
}

// Function with no return value
void printMessage(const char *message) {
    printf("%s\n", message);
    // No return statement needed
}

Function Overloading

C does not support function overloading (having multiple functions with the same name but different parameters). This is a feature available in C++ but not in C.

Recursive Functions

A recursive function is a function that calls itself. Recursive functions need a base case to terminate the recursion.

// Calculate factorial using recursion
int factorial(int n) {
    // Base case
    if (n == 0 || n == 1) {
        return 1;
    } 
    // Recursive case
    else {
        return n * factorial(n - 1);
    }
}

int main() {
    printf("Factorial of 5: %d\n", factorial(5));
    return 0;
}
Factorial of 5: 120

Trace of factorial(5):

factorial(5) = 5 * factorial(4)
            = 5 * (4 * factorial(3))
            = 5 * (4 * (3 * factorial(2)))
            = 5 * (4 * (3 * (2 * factorial(1))))
            = 5 * (4 * (3 * (2 * 1)))
            = 5 * (4 * (3 * 2))
            = 5 * (4 * 6)
            = 5 * 24
            = 120

Static Functions

A static function is accessible only within the file in which it is defined. It cannot be called from other files.

// Static function definition
static int privateFunction(int x) {
    return x * 2;
}

Inline Functions

An inline function is a function that is expanded in line when it is called. This can improve performance by eliminating function call overhead.

// Inline function (C99 and later)
inline int max(int a, int b) {
    return (a > b) ? a : b;
}

Function Pointers

Function pointers are variables that store the address of functions. They can be used to implement callbacks and create more flexible code.

// Function pointer declaration
int (*operation)(int, int);

// Functions to be pointed to
int add(int a, int b) { return a + b; }
int subtract(int a, int b) { return a - b; }

int main() {
    // Assign function address to pointer
    operation = add;
    printf("Result of add: %d\n", operation(5, 3));
    
    // Change function pointer
    operation = subtract;
    printf("Result of subtract: %d\n", operation(5, 3));
    
    return 0;
}
Result of add: 8
Result of subtract: 2

Variable-Length Argument Lists

C supports functions with a variable number of arguments using the <stdarg.h> header.

#include <stdarg.h>

// Function to calculate the sum of a variable number of integers
int sum(int count, ...) {
    va_list args;
    int total = 0;
    
    // Initialize the argument list
    va_start(args, count);
    
    // Access all arguments
    for (int i = 0; i < count; i++) {
        total += va_arg(args, int);
    }
    
    // Clean up
    va_end(args);
    
    return total;
}

int main() {
    printf("Sum(2, 5, 10): %d\n", sum(2, 5, 10));
    printf("Sum(4, 1, 2, 3, 4): %d\n", sum(4, 1, 2, 3, 4));
    return 0;
}
Sum(2, 5, 10): 15
Sum(4, 1, 2, 3, 4): 10

Best Practices for Functions

  • Each function should perform a single, well-defined task
  • Use meaningful function names that describe what the function does
  • Keep functions short and focused (usually less than 20-30 lines)
  • Document functions with comments explaining their purpose, parameters, and return values
  • Validate function inputs to handle edge cases and prevent errors
  • Place function prototypes in header files for better organization

Example: Complete Program with Functions

// Program to calculate the area and perimeter of a rectangle
#include <stdio.h>

// Function prototypes
float calculateArea(float length, float width);
float calculatePerimeter(float length, float width);
void displayResults(float length, float width, float area, float perimeter);

int main() {
    float length, width;
    
    // Get input from user
    printf("Enter length of rectangle: ");
    scanf("%f", &length);
    
    printf("Enter width of rectangle: ");
    scanf("%f", &width);
    
    // Calculate area and perimeter
    float area = calculateArea(length, width);
    float perimeter = calculatePerimeter(length, width);

Arrays

Arrays are collections of elements of the same data type stored in contiguous memory locations. They allow you to store multiple values under a single variable name.

Array Declaration and Initialization

To declare an array, specify the data type of its elements, its name, and the size (number of elements):

// Array declaration
data_type array_name[size];

// Examples
int numbers[5];           // An array of 5 integers
float temperatures[100];  // An array of 100 floating-point values
char name[20];            // An array of 20 characters

You can initialize an array at the time of declaration:

// Initialize with values
int numbers[5] = {10, 20, 30, 40, 50};

// Initialize with fewer values than the size
int partial[5] = {10, 20, 30};  // Remaining elements are initialized to 0

// Let the compiler determine the size
int auto_size[] = {5, 10, 15, 20, 25};  // Size is 5

// Initialize all elements to zero
int zeros[10] = {0};

Accessing Array Elements

Array elements are accessed using indices, which start at 0:

int numbers[5] = {10, 20, 30, 40, 50};

// Accessing elements
printf("First element: %d\n", numbers[0]);  // 10
printf("Third element: %d\n", numbers[2]);  // 30

// Modifying an element
numbers[1] = 25;
printf("Modified second element: %d\n", numbers[1]);  // 25
First element: 10
Third element: 30
Modified second element: 25

Array Traversal

You can traverse an array using loops:

int numbers[5] = {10, 20, 30, 40, 50};

// Using a for loop to display all elements
printf("Array elements: ");
for (int i = 0; i < 5; i++) {
    printf("%d ", numbers[i]);
}
printf("\n");

Multi-dimensional Arrays

// Declaration of a 2D array
int matrix[3][4];  // 3 rows, 4 columns

int grid[3][3] = {

Arrays and Functions

When an array is passed to a function, what actually gets passed is the memory address of the first element (not a copy of the array):

// Function to calculate the sum of array elements
int calculateSum(int arr[], int size) {
    int sum = 0;
    for (int i = 0; i < size; i++) {
        sum += arr[i];
    }
    return sum;
}

int main() {
    int numbers[5] = {10, 20, 30, 40, 50};
    int result = calculateSum(numbers, 5);
    printf("Sum: %d\n", result);
    return 0;
}

Note: You need to pass the size of the array separately because the function doesn't inherently know the array's size.

Limitations and Best Practices

  • Array size must be specified at compile time (fixed size)
  • C doesn't check array bounds - accessing elements outside the array causes undefined behavior
  • Arrays cannot be assigned to other arrays directly (use loops or functions like memcpy)
  • Use symbolic constants for array sizes to make code more maintainable
  • Always initialize arrays before using them
  • Be careful when passing arrays to functions, as changes inside the function affect the original array

Pointers

Pointers are variables that store memory addresses. They are one of the most powerful features of C, allowing direct memory manipulation, dynamic memory allocation, and efficient parameter passing.

Pointer Basics

Declaration and Initialization

// Pointer declaration
data_type *pointer_name;

// Examples
int *ptr;       // Pointer to an integer
char *cptr;     // Pointer to a character
float *fptr;    // Pointer to a float

To initialize a pointer, assign it the address of another variable using the address-of operator (&):

int num = 10;
int *ptr = #  // ptr now holds the address of num

Accessing Values through Pointers

The dereference operator (*) is used to access the value stored at the address contained in a pointer:

int num = 10;
int *ptr = #

printf("Value of num: %d\n", num);
printf("Address of num: %p\n", &num);
printf("Value stored in ptr: %p\n", ptr);
printf("Value pointed to by ptr: %d\n", *ptr);

// Modify the value using the pointer
*ptr = 20;
printf("New value of num: %d\n", num);

Address of num: 0x7ffd5c9f2a4c
Value stored in ptr: 0x7ffd5c9f2a4c
Value pointed to by ptr: 10
New value of num: 20

Pointer Arithmetic

C allows arithmetic operations on pointers, which is useful when working with arrays and other data structures:

int numbers[5] = {10, 20, 30, 40, 50};
int *ptr = numbers;  // ptr points to the first element

// Access array elements using pointer arithmetic
printf("First element: %d\n", *ptr);
printf("Second element: %d\n", *(ptr + 1));
printf("Third element: %d\n", *(ptr + 2));

// Incrementing a pointer
ptr++;  // ptr now points to the second element
printf("After increment, ptr points to: %d\n", *ptr);
First element: 10
Second element: 20
Third element: 30
After increment, ptr points to: 20

Important notes about pointer arithmetic:

  • When you add 1 to a pointer, it actually increases by the size of the data type it points to
  • For an int pointer, ptr + 1 adds 4 bytes (assuming int is 4 bytes)
  • For a char pointer, ptr + 1 adds 1 byte

Pointers and Functions

Pass by Reference using Pointers

Pointers allow functions to modify variables from the calling function (pass by reference):

// Function to swap two integers using pointers
void swap(int *a, int *b) {
    int temp = *a;
    *a = *b;
    *b = temp;
}

int main() {
    int x = 5, y = 10;
    printf("Before swap: x = %d, y = %d\n", x, y);
    
    swap(&x, &y);  // Pass addresses of x and y
    
    printf("After swap: x = %d, y = %d\n", x, y);
    return 0;
}

After swap: x = 10, y = 5

Array Parameters as Pointers

When you pass an array to a function, it's actually passed as a pointer:

    for (int i = 0; i < size; i++) {

Dynamic Memory Allocation

Pointers are used with dynamic memory allocation functions to create variables and arrays whose size is determined at runtime:

#include <stdlib.h>  // Required for memory allocation functions

// Allocate a single integer
int *p = (int*)malloc(sizeof(int));
*p = 10;
printf("Value: %d\n", *p);

// Allocate an array of integers
int n = 5;
int *arr = (int*)malloc(n * sizeof(int));

// Initialize the array
for (int i = 0; i < n; i++) {
    arr[i] = i * 10;
}

// Display the array
printf("Dynamic array: ");
for (int i = 0; i < n; i++) {
    printf("%d ", arr[i]);
}
printf("\n");

// Free memory when done
free(p);
free(arr);
Value: 10
Dynamic array: 0 10 20 30 40

Common dynamic memory functions:

  • malloc(size): Allocates specified bytes of memory
  • calloc(n, size): Allocates memory for n elements of size bytes each and initializes to zero
  • realloc(ptr, new_size): Changes the size of previously allocated memory
  • free(ptr): Releases previously allocated memory

Common Pointer Pitfalls

  • Uninitialized pointers: Using a pointer before it's initialized can cause unpredictable behavior
  • Dangling pointers: Pointers that reference memory that has been freed or is out of scope
  • Memory leaks: Forgetting to free dynamically allocated memory
  • Buffer overflows: Writing beyond the allocated memory
  • Null pointer dereferencing: Trying to access the value at a NULL pointer
// Example of proper NULL checking
int *ptr = malloc(sizeof(int));
if (ptr == NULL) {
    printf("Memory allocation failed\n");
    return 1;
}

// Use pointer
*ptr = 42;
printf("Value: %d\n", *ptr);

// Free memory
free(ptr);
ptr = NULL;  // Good practice to prevent dangling pointer

In C, strings are arrays of characters terminated by a null character ('\0'). C doesn't have a dedicated string data type, so character arrays and pointers are used to handle strings.

2. Removing Whitespace

// Function to trim leading and trailing whitespace
char* trim(char* str) {
    if (str == NULL) return NULL;
    
    char* end;
    
    // Trim leading space
    while (isspace((unsigned char)*str)) str++;
    
    if (*str == 0) return str;  // All spaces
    
    // Trim trailing space
    end = str + strlen(str) - 1;
    while (end > str && isspace((unsigned char)*end)) end--;
    
    // Write new null terminator
    *(end + 1) = '\0';
    
    return str;
}

int main() {
    char text[] = "   Hello, World!   ";
    printf("Before: \"%s\"\n", text);
    
    char* trimmed = trim(text);
    printf("After: \"%s\"\n", trimmed);
    
    return 0;
}
Before: " Hello, World! "
After: "Hello, World!"

#include <stdio.h>
#include <stdlib.h>
#include <string.h>
#include <ctype.h>

// Convert string to title case (capitalize first letter of each word)
void toTitleCase(char* str) {
    int capitalize = 1;  // Flag to indicate if the next character should be capitalized
    
    while (*str != '\0') {
        if (isspace(*str)) {
            capitalize = 1;
        } else if (capitalize) {
            *str = toupper(*str);
            capitalize = 0;
        } else {
            *str = tolower(*str);
        }
        str++;
    }
}

// Count occurrences of a substring in a string
int countSubstring(const char* str, const char* sub) {
    int count = 0;
    const char* temp = str;
    
    while ((temp = strstr(temp, sub)) != NULL) {
        count++;
        temp += strlen(sub);
    }
    
    return count;
}

// Join an array of strings with a delimiter
char* joinStrings(const char** strings, int count, const char* delimiter) {
    // Calculate total length
    size_t total_len = 0;
    size_t delim_len = strlen(delimiter);
    
    for (int i = 0; i < count; i++) {
        total_len += strlen(strings[i]);
        if (i < count - 1) {
            total_len += delim_len;
        }
    }
    
    // Allocate memory for the result
    char* result = (char*)malloc(total_len + 1);  // +1 for null terminator
    if (result == NULL) return NULL;
    
    // Join strings
    result[0] = '\0';  // Start with empty string
    for (int i = 0; i < count; i++) {
        strcat(result, strings[i]);
        if (i < count - 1) {
            strcat(result, delimiter);
        }
    }
    
    return result;
}

    printf("Count of 'the': %d\n", countSubstring(sentence, "the"));
    printf("Count of 'fox': %d\n\n", countSubstring(sentence, "fox"));
    
    // Test joinStrings
    const char* words[] = {"Hello", "World", "of", "C", "Programming"};

Summary of String Handling in C

String handling in C requires careful attention to memory management and buffer sizes. Key points to remember:

  • Strings in C are null-terminated character arrays
  • Always ensure there's enough space for the null terminator
  • The standard library provides many functions for string manipulation in <string.h>
  • Use length-limited functions whenever possible to prevent buffer overflows
  • Be careful with dynamic memory allocation for strings—always free memory when done
  • String parsing requires careful handling of tokens and boundaries
  • For internationalization, consider wide character support or external libraries

With a strong understanding of arrays, pointers, and string handling, you can build robust and efficient C programs that handle textual data effectively and safely.
