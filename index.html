Hello World Example

#include <stdio.h>

int main() {
    // This is a comment
    printf("Hello, World!\n");
    return 0;
}

Output: Hello, World!

Variables & Data Types

#include <stdio.h>

int main() {
    // Integer variable
    int age = 25;

Age: 25
Height: 5.9
Grade: A

Control Flow - If-Else

#include <stdio.h>

int main() {
    int number = 10;
    
    if (number > 0) {
        printf("Number is positive.\n");
    } else if (number < 0) {
        printf("Number is negative.\n");
    } else {
        printf("Number is zero.\n");
    }
    
    return 0;
}

Output: Number is positive.

Introduction to C Programming

C is a powerful general-purpose programming language that was originally developed by Dennis Ritchie between 1969 and 1973 at Bell Labs. It's one of the most widely used programming languages of all time and has influenced many other popular languages like C++, Java, and JavaScript.

C is known for its efficiency, flexibility, and control. It's often referred to as a "middle-level" language because it combines elements of high-level languages with the functionality of assembly language.

Setting Up Your Environment

Before you can start writing C programs, you need to set up a development environment. Here's how you can do it on different operating systems:

Windows

  1. Download a C compiler like MinGW or Microsoft Visual Studio Community.
  2. Install the compiler following the installation instructions.
  3. Set up environment variables for easy access to the compiler.
  4. (Optional) Install an IDE like Code::Blocks or VS Code for a better development experience.

macOS

  1. Open Terminal and install Xcode Command Line Tools by running: xcode-select --install
  2. This will install the GCC compiler which includes support for C programming.
  3. (Optional) Install an IDE like VS Code for a better development experience.

Linux

  1. Open Terminal and install GCC by running: sudo apt-get install gcc (for Ubuntu/Debian) or sudo yum install gcc (for Fedora/RHEL).
  2. (Optional) Install an IDE like VS Code for a better development experience.
For beginners, we recommend using an online compiler like C Compiler Online if you want to quickly start writing code without setting up a local environment.

Your First C Program

Now that your environment is set up, let's write your first C program:

hello.c

#include <stdio.h>

int main() {
    // This is a comment
    printf("Hello, C Programming!\n");
    return 0;
}

How to Compile and Run

  1. Save the code above in a file named hello.c
  2. Open your terminal or command prompt
  3. Navigate to the directory where you saved the file
  4. Compile the program: gcc hello.c -o hello
  5. Run the program:
    • On Windows: hello
    • On macOS/Linux: ./hello

You should see the output: Hello, C Programming!

Basic Syntax

Let's break down the basic syntax elements from our hello world program:

1. Header Files

#include <stdio.h> - This line includes the Standard Input/Output header file which contains functions like printf() for input and output operations.

2. The main() Function

int main() { ... } - Every C program starts execution from the main function. The int before main indicates that this function returns an integer value.

3. Comments

// This is a comment - Comments are notes in the code that are ignored by the compiler. They help make the code more readable.

4. Statements and Semicolons

printf("Hello, C Programming!\n"); - This statement prints text to the console. Note that each statement ends with a semicolon (;).

5. Return Statement

return 0; - This returns a value of 0 to the operating system, indicating that the program executed successfully.

Variables & Data Types

Variables are used to store data in a program. In C, you must declare a variable's type before using it.

Basic Data Types

  • int: Integer values (e.g., 1, -3, 42)
  • float: Single-precision floating-point values (e.g., 3.14, -2.5)
  • double: Double-precision floating-point values (e.g., 3.141592653589793)
  • char: Single characters (e.g., 'A', '7', '$')

Variable Declaration and Initialization

variables.c

#include <stdio.h>

int main() {
    // Variable declarations
    int age;            // Declaration
    int height = 175;   // Declaration and initialization
    
    // Assigning a value
    age = 30;
    
    // Multiple variables of the same type
    float weight = 68.5, bmi = 22.4;
    
    // Character variable
    char grade = 'A';
    
    // Printing variables
    printf("Age: %d years\n", age);
    printf("Height: %d cm\n", height);
    printf("Weight: %.1f kg\n", weight);
    printf("BMI: %.1f\n", bmi);
    printf("Grade: %c\n", grade);
    
    return 0;
}

Operators in C

Operators are symbols that tell the compiler to perform specific mathematical or logical operations.

Arithmetic Operators

  • +: Addition
  • -: Subtraction
  • *: Multiplication
  • /: Division
  • %: Modulus (remainder after division)

Example

operators.c

#include <stdio.h>

int main() {
    int a = 10, b = 3;
    
    printf("a + b = %d\n", a + b);   // Addition
    printf("a - b = %d\n", a - b);   // Subtraction
    printf("a * b = %d\n", a * b);   // Multiplication
    printf("a / b = %d\n", a / b);   // Division (integer division)
    printf("a %% b = %d\n", a % b);  // Modulus (remainder)
    
    // Floating-point division
    float result = (float)a / b;
    printf("a / b (float) = %.2f\n", result);
    
    return 0;
}

Frequently Asked Questions

Find answers to common questions about learning C programming

C has a steeper learning curve compared to some modern languages like Python, but it's definitely learnable with the right resources. Our structured approach makes it accessible for beginners while still covering advanced concepts for experienced programmers. The key is consistent practice and hands-on coding.

No prior programming experience is required! Our C programming course starts from absolute basics and gradually builds up to more complex concepts. However, if you do have experience with other programming languages, you'll likely progress faster through the initial sections.

C remains incredibly relevant despite being developed in the 1970s. It's widely used in operating systems, embedded systems, game development, and high-performance applications. Learning C gives you a deeper understanding of how computers work at a lower level, making you a better programmer overall. Additionally, many modern languages like C++, Java, and JavaScript have syntax influenced by C.

The time required varies based on your dedication and prior experience. Beginners can grasp the fundamentals in 2-3 weeks with consistent practice. Achieving proficiency might take 3-6 months of regular coding and completing projects. Our structured curriculum is designed to guide you through this journey efficiently.

With C, you can build a wide range of applications including operating systems, embedded systems, game engines, databases, compilers, desktop applications, and system utilities. Many critical infrastructure software like Linux kernel, Windows components, database engines like MySQL and PostgreSQL, and game engines like Unity (core components) are written in C.
